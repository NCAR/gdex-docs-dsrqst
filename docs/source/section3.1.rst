3.1 - Request Control Actions
=================================

A request control record for a dataset, and optionally a group index, is
the starting point for placing a dataset or group under **dsrqst** control
for a specified request type.

Request control information can be created, modified, and viewed through
the actions in this section:

.. list-table::
   :widths: auto
   :header-rows: 0

   * - :ref:`Set Request Control Information <section3.1.1>`
     - create or modify request control information for specified datasets, or for specific group indices, of a request type
   * - :ref:`Get Request Control Information <section3.1.2>`
     - retrieve information for existing request controls
   * - :ref:`Delete Request Control Information <section3.1.3>`
     - delete one or more request control records
   * - Get Source File information
     - retrieve information about included source files
   * - Set Source File information
     - add or modify information about included source files
   * - Delete Source File Information
     - delete one or more included source-file records of a specified request control
